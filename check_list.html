🎯 블록코딩 챗봇 평가 기준표

1. 정보 정확성 (Information Accuracy) - 25점
평가 항목 배점 점수 기준 예시
블록 이름 10점 10점 정확한 공식 명칭 사용: "시작하기 버튼을 클릭했을 때"
8-9점 거의 정확: "시작하기 버튼을 클릭했을 때 블록"
5-7점 의미 통하나 부정확: "시작 블록", "실행 블록"
1-4점 잘못된 이름: "플레이 블록", "런 블록"
0점 블록 이름 미제시
카테고리 및 색상 10점 10점 카테고리명 + 정확한 색상: "시작 카테고리 (초록색)"
7-8점 카테고리만 정확: "시작 카테고리"
3-5점 부분 정확: "초록색 카테고리"
1-2점 잘못된 정보: "시작 카테고리 (노란색)"
0점 정보 미제시
설명 정확도 5점 5점 기능과 용도 정확히 설명
3-4점 기본 기능만 설명
0-2점 설명 없거나 오류
2. 시각적 구조화 (Visual Structure) - 25점
평가 항목 배점 점수 기준 예시
정보 계층화 15점 15점 카드/섹션으로 명확히 구분
10-12점 부분적 구조화 (제목, 본문 구분)
5-9점 기본 문단 구분
0-4점 연속된 텍스트
시각적 단서 10점 10점 아이콘 + 색상 + 강조 표시
7-8점 2가지 요소 사용
4-6점 1가지 요소만 사용
0-3점 시각적 요소 없음
